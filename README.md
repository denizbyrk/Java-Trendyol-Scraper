# Java Trendyol Scraper - Java & JSoup

This application allows the user to fetch a product's data that is on Trendyol in just a few seconds.

The following data can be fetched from the product:
-  Name
-  Price
-  Description
-  Seller
-  Rankings
-  Favourites
-  Category
-  Properties
-  Comments
